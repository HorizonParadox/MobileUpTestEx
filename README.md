# MobileUpTestEx
Test task for MobileUp

Декомпозиция (Хронологически):
  1. Анализ макета и API: 30 минут.
  2. Вёрстка макета: 4 часов.
  3. Настройка Retrofit, тестирование запросов к API: 3 часа.
  4. Теоретическое изучение Dagger2/Hilt: 2 часа.
  5. Настройка DI: 3-4 часа.
  6. Создание RecyclerView, создание макета элемента, вставка данных, отладка обновляемых данных : 2 часа. 
  7. Доработка UI (ProgressBar, ErrorLayout): 1 час.
  8. Добавление Fragment Navigation: 30 минут.
  9. Исправление багов, "чистка" кода: 3 часа.
